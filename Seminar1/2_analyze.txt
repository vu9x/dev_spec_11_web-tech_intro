Структура страницы сайта https://ru.wikipedia.org/
1. Header \ Шапка
2. Content \ Контент
3. Навигация
4. Footer \ Подвал

1. Header \ Шапка
Состоит из следующих элементов:
* Текущего пользователя:
    _Регистрация и Вход
    _Вклад и обсуждение (проведенные данным пользователем)
* Относящихся к сайту:
    Вкладки:
    _Заглавная - основной контент сайта
    _Обсуждения - Здесь обсуждается только сама заглавная страница и включённые в неё шаблоны.

    Фильтры:
    _читать - Основной контент для пользователей
    _Просмотр кода - показывает html код текущей страницы, где сидит пользователь
    _История - история изменения текущей страницы
    _Поисковая строка

2. Content \ Контент
* Блок приветствия на всю страницу, которая содержит следующую информацию:
    _Определение Википедии
    _Кол-во пользователей Вики на локальном рынке\стране
    Кнопки:
    _Порталы - ссылка на категории контента 
    _Справка
    _Создать статью
* Блок статей: Текстографические блоки состоящие из:
Маленькой картинки в верхнем правом углу каждого блока. 
Название блока.
Заголовка статьи. 
Текста статьи (короткое превью) с гиперссылками на другие статьи.
Каждый блок кликабельный. Позволяет зайти в саму статью для получения более подробной информации.
Блок в подвале содержит кнопку читать, возможно количество статьей по данной категории, доп меню по блоку.
Блоки на главной странице Википедия:
    _Избранная статья
    _Из новых материалов
    _Хорошая статья
    _Текущие события
    _В этот день
    _и прочие

3. Навигация
Навигация занимает полностью левую сторону сайта и состоит из следующих блоков\ссылок на разделы:
* Логотипа сайта
* Общий блок статей: Заглавная, Содержание, Статьи(Избранные, Случайные, Текущие), Пожертвовать
* Участие - разделы связанные с написанием\редактированием статей
* Инструменты 
* Печать\Экспорт - возможность сохранить страницу\статью на компьютер пользователя
* В других проектах - другие проекты Википедии
* На других языках - Википедия на других языках

4. Footer \ Подвал
Содержит следующую информацию:
* Лицензии и условия использования
* Политика конфиденциальности
* Описание Википедии
* Отказ от ответственности
* Свяжитесь с нами 
* Кодекс поведения
* Мобильная версия
* Разработчики
* Статистика
* Ссылки на владельцев Вики
